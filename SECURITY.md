# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.9   | :white_check_mark: |
| < 0.1.8 | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities via GitHub's private vulnerability reporting feature or email pasteproof@gmail.com.

Do not open public issues for security vulnerabilities.

We aim to respond within 48 hours and will work with you on coordinated disclosure.
